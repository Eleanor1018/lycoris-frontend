# 关于夏水仙

欢迎来到 夏水仙 Lycoris 🌸

这是一个为跨性别者提供无障碍设施信息和互助信息的平台 💗

夏水仙由3个部分组成：
- 地图：夏水仙的核心功能，我们可以在地图上标注无障碍卫生间、跨性别友好场所等地点，实现信息共享
- 文档：经验分享，例如HRT指南、穿搭经验；尽量用最简洁的语言、最容易理解的方式把行之有效的方案/踩过的坑分享给大家
- 工具（开发中）：隐私优先的健康管理工具将以独立 App 形式上线（如服药提醒、情绪与体征记录）

夏水仙由 雪雁 Eleanor 设计并运营，如果有建议、遇到问题，请联系：LycorisTrans@gmail.com

下面我想分享有关于自己的故事，也是创建这个项目的初衷：

“小姑娘，这里是男卫生间！”又一次，我被热心的大叔叫住。  
应该开心吗？我被认成了女孩子，被人友善的对待，这说明我至少有一点点pass；应该难过吗？我没有手术，身份证也没有改，在社会意义上，我仍然是男性。  
我尴尬地笑了笑，甚至不敢和大叔说“谢谢”，因为我半成品的伪音只会吓到路人；我只好跑出去，趁着没人的时候去卫生间，或者跑回家...我总是因此徘徊而踌躇着，我不想吓到普通的路人，更不想自己被审视，没有第三卫生间的时候，我总会遇到这样的困难。  
出门时，我戴上口罩和帽子，把自己遮得严严实实的，以此来逃避人们的目光和社会的批判。我总是逃避着，躲在阴影里隐藏自己。

但是，这世界上有千千万万个像我一样的跨性别者，他们都有权利生活在阳光下，自然的展示自己，不必躲闪，不必隐藏，堂堂正正，大大方方。每一个人都应当有平等的权益，跨性别者也不例外。

社会的进步是缓慢的，然而每一个微小的努力都会被看见。就从无障碍卫生间开始。  
我们会搜集很多个无障碍卫生间，只要打开手机，就可以看到哪里可以上厕所。我们也会搜集很多地点，能买到国补的药房，能开药的内分泌科，友好的精神科。我们还会搜集，哪里有廉价的食物，有安全免费的过夜地点，甚至是扭转机构的位置。

同时，我们会把我们的经验写成文档，分享给后来的跨性别者。当然，这些文档不是“维基百科”式的说明文，那些已经有前人做过了。我们是想要分享踩过的坑、需要注意的危险，并且用简明的语言构筑行之有效的方案。

感谢您的驻足阅读，如果夏水仙/Lycoris能帮到您，我会非常开心！

by 雪雁
